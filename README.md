# Predicting 30-Day Hospital Readmissions for Diabetic Patients

## Project Overview

Hospital readmissions are a major challenge for healthcare systems, affecting patient outcomes and increasing healthcare costs. This project develops machine learning models to predict whether a diabetic patient will be readmitted within 30 days of discharge using the Diabetes 130-US Hospitals dataset.

The project follows an end-to-end data science workflow, including data exploration, preprocessing, feature engineering, model development, evaluation, and explainable AI techniques.

---

## Objectives

- Analyze factors associated with hospital readmissions.
- Build predictive models for 30-day readmission risk.
- Compare multiple machine learning algorithms.
- Interpret model predictions using explainable AI.
- Provide insights that could support clinical decision-making.

---

## Dataset

**Dataset:** Diabetes 130-US Hospitals Dataset

- Approximately 100,000 patient encounters
- 130 hospitals across the United States
- Multiple demographic, diagnostic, and treatment variables
- Readmission outcomes
